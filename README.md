ShelfStack Project

This repository contains the implementation of the ShelfStack task, organized into two distinct branches based on the programming language used.
📂 Repository Structure & Branches

The project is split into two main branches to showcase both JavaScript and TypeScript implementations:

    javascript branch:

        Contains the full implementation using pure JavaScript (ES6+).

        Focuses on functional logic and standard JS syntax.

    typescript branch:

        Contains the full implementation using TypeScript.

        Includes tsconfig.json, source files (.ts), and type definitions.

🚀 How to Run the Project
1. JavaScript Version

Follow these steps to run the JS implementation:

    Switch Branch: Move to the javascript branch using git checkout javascript.

    Execute: Run the file directly using Node.js with node shelfstack.js.

2. TypeScript Version

Follow these steps to run the TS implementation:

    Switch Branch: Move to the typescript branch using git checkout typescript.

    Install Dependencies: Setup the environment by running npm install.

    Compile: Convert TypeScript to JavaScript using npx tsc.

    Execute: Run the generated file with node shelfstack.js.

🛠 Prerequisites

    Node.js: Ensure Node.js is installed on your machine.

    npm: Required to manage packages and the TypeScript compiler.

# ShelfStack Project

This repository contains the implementation of the **ShelfStack** task, organized into two distinct branches based on the programming language used.

## 📂 Repository Structure & Branches

The project is split into two main branches to showcase both JavaScript and TypeScript implementations:

* **`javascript` branch**: 
    - Contains the full implementation using pure **JavaScript (ES6+)**.
    - Focuses on functional logic and standard JS syntax.
* **`typescript` branch**: 
    - Contains the full implementation using **TypeScript**.
    - Includes `tsconfig.json`, source files (`.ts`), and type definitions.

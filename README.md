# ShelfStack Project

This repository contains the implementation of the ShelfStack task in two different versions, organized into separate branches:

## Branches Structure:
- **[javascript branch]**: Contains the full implementation using pure JavaScript.
- **[typescript branch]**: Contains the full implementation using TypeScript, including `tsconfig.json` and source files.

## How to Run:
1. **JavaScript Version**: Switch to the `javascript` branch and run `node shelfstack.js`.
2. **TypeScript Version**: Switch to the `typescript` branch, run `npm install` (if needed), then compile using `tsc`.
